# FocusFlow (ポモドーロ・タイマー)

自分が日々の勉強や開発で「本当に使いたくなる」を目指して開発している、シンプルで集中力を高めるためのポモドーロ・タイマーアプリです。WindowsとMacのクロスプラットフォームで開発を行っています。

## 🚀 特徴・こだわりたいポイント (UI/UX)
- **圧倒的なシンプルさ**: 勉強中に余計なノイズを入れない、洗練されたデザイン。
- **直感的な操作性**: 迷わずタイマーを開始でき、集中モードに入り込めるUX。
- **マルチデバイス開発**: Git/GitHubを活用し、WindowsとMacの両環境でシームレスに開発を継続。

## 🛠️ 開発環境 / テクノロジー
- **言語**: HTML5, CSS3, JavaScript (Vanilla JS)
- **実行環境**: Node.js
- **ツール**: Vite (ローカル開発サーバー)
- **バージョン管理**: Git / GitHub

## 🏃 起動方法 (ローカル再現手順)

別の環境や、PCを新しくした際にこのアプリを手元で動かすための手順です。

```bash
# 1. リポジトリをクローン（またはフォルダを開く）
# 2. 必要な部品のインストール
npm install

# 3. 開発サーバーの起動
npm run dev
